HOW TO TEST LOCALLY
1. Keep all files in the same folder.
2. Double-click index.html to open it in a browser.
3. Click either download button to confirm the Excel file downloads.

HOW TO PUBLISH
- Upload all files to the same folder on your existing website.
- Deploy the folder as a static site through Vercel, Netlify, GitHub Pages, or similar hosting.
- Do not rename the Excel file unless you also update the two download links in index.html.
